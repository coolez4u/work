🚫

경쟁사의 우리 IP 차단

이한씨앤씨(ehancnc.co.kr)와 컨스트렉션(blog.naver.com/rain7193)은 인천 청라 지역 IP를 광고 노출에서 차단해 두었습니다. 이로 인해 당사 담당자가 "실내건축면허" 등 주요 키워드를 검색할 때 두 업체의 광고가 노출되지 않았습니다.

🔢

가상 1위 착시 현상

담당자 화면에서는 디에이치·바로건설 광고가 1~2위로 표시되었으나, 실제 다른 지역 검색 결과에서는 이한, 컨스트렉션이 1~2위를 점유하고 있었습니다. 즉, 우리 광고의 실제 순위는 3~5위 수준이었습니다.

💸

입찰가 과소 설정

담당자 기준 1위를 유지하는 입찰가(약 8,000원)로 세팅되었지만, 실제 시장에서 2~3위를 유지하기 위해서는 10,000~13,000원 수준이 필요했습니다. 결과적으로 입찰 경쟁에서 낮은 순위를 차지하면서도 더 높은 CPC를 지불하는 구조가 형성되었습니다.

📈

실질 CPC 격차 발생

기존 계획 대비 클릭당 1,000~3,000원의 추가 비용이 발생하였으며, 이는 동일한 클릭 수 대비 광고비 소진을 빠르게 가속시킨 직접 원인입니다. 시간대별로도 경쟁 강도 차이로 인해 CPC 편차가 발생하였습니다.

개선 방안 및 조치 계획
IP 차단 문제 해소 및 입찰가 재정비
우선순위 조치 항목 상세 내용 기대 효과
긴급 VPN 활용 순위 검증 주요 키워드(실내건축면허, 종합건설면허, 기업진단 등)를 타지역 IP로 정기적으로 모니터링하여 실제 노출 순위 확인 실제 경쟁 환경 파악 → 정확한 입찰가 설정
긴급 주요 키워드 입찰가 재조정 실내건축면허 8,000원 → 11,000~13,000원으로 상향 조정 (2~3위 진입 목표) 실제 2~3위 유지 → 클릭률 개선 기대
중요 시간대별 입찰 조정 경쟁이 강한 오전 9~11시, 오후 2~4시대 입찰가 별도 세팅 또는 자동 입찰 전략 도입 검토 CPC 안정화 및 불필요한 고가 클릭 방지
중요 광고비 일일 한도 재설정 현재 무제한(또는 높은 한도) → 일별 한도 설정으로 소진 속도 관리 월 예산 예측 가능성 향상
권고 경쟁사 IP 차단 역활용 경쟁사가 우리 IP를 차단했으므로, 동일하게 당사도 주요 경쟁사 IP 대역 차단 여부 검토 불필요한 경쟁사 클릭 방지
권고 정기 리포트 체계 수립 주 1회 타지역 IP 기준 순위 스크린샷 비교 리포트 운영. 이번 사례 재발 방지 프로세스 수립 광고 운영 투명성 및 정확도 향상
⚠️ 이번 문제의 본질

광고 세팅 담당자의 검색 환경(청라 지역 IP)에서 경쟁사 광고가 차단되어 있어 실제 경쟁 강도를 크게 낮게 인식한 것이 근본 원인입니다. 이는 단순 실수가 아닌, 경쟁사들이 의도적으로 설계한 구조임을 인지해야 합니다.

✅ 조기 발견의 의의

VPN 활용 비교 검색을 통해 이 문제를 확인한 것은 매우 중요한 성과입니다. 향후 동일한 방법으로 최소 월 2회 순위 점검을 통해 입찰가 최적화 및 광고비 효율을 관리할 것을 권장합니다.

7
문제 발생 타임라인 요약
광고 집행 → 이상 감지 → 원인 분석 → 개선

광고 세팅 초기

청라 IP 환경에서 경쟁사(이한, 컨스트렉션) 광고가 차단되어 보이지 않음. 담당자 화면 기준 1위 유지를 위한 입찰가(~8,000원)로 세팅 완료.

6월 8일~11일 (기간 A)

총 광고비 1,931,981원, 평균 CPC 3,000원. 충전 주기 약 4~5일 수준 유지. 표면상 정상 운영으로 인식.

이상 징후 감지

광고비 소진 주기가 기존 4~5일 → 3~4일로 단축되는 현상 발견. 동일 예산 대비 클릭수 및 광고비 비례가 맞지 않는 현상 인지.

6월 15일~18일 (기간 B)

총 광고비 2,268,572원, 평균 CPC 3,133원으로 상승. 타지역 IP 확인 결과 경쟁사(이한, 컨스트렉션)가 실제 1~2위를 점유하고 있음을 확인.

원인 규명

경쟁사들이 청라 IP를 차단하여 담당자 환경에서는 보이지 않았던 것으로 분석. 실제 2~3위 유지를 위해서는 10,000~13,000원 입찰가 필요한 시장임을 확인.

개선 조치 계획 수립

VPN 기반 정기 순위 모니터링 체계 도입, 주요 키워드 입찰가 상향 조정, 일일 예산 한도 세팅 등 후속 조치 실행 예정.
